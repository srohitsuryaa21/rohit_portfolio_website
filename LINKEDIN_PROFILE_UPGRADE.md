# LinkedIn Profile Upgrade — Rohit Suryaa Saravanan

This is the recommended LinkedIn profile setup based on the current portfolio, target roles, and current market expectations for software/data candidates.

Certificates are intentionally excluded because they will be added later.

## Strategy

Your LinkedIn should not read like a student profile only. It should position you as an early-career technical builder who can work across:

- C# / .NET software development
- Backend engineering
- Data engineering
- Data science
- Machine learning
- Applied research

The core message should match the portfolio:

> I build reliable software and intelligent data systems.

That line is suitable because it works across both software and data roles. It is broader than “data scientist” and stronger than “student looking for opportunities.”

Use this as the primary headline:

```text
Software Developer | C#/.NET & Backend Engineering | Data Science & ML | MSc Data Science @ Hochschule Fulda | Building reliable software and intelligent data systems
```

Alternative shorter version:


## Sources used for direction

- LinkedIn Help, “Create a good LinkedIn profile”: complete profile, professional photo, background image, rewritten headline, About summary, relevant work/education, media samples, relevant skills, recommendations, and custom public URL.
- TIME, “6 Things Recruiters Look For in Your LinkedIn Profile”: recruiters scan photo, headline, career history, connections, education, endorsements, and recommendations; the headline should describe who you are professionally rather than saying only that you are seeking opportunities.
- Dorn et al., “Multidimensional Skills as a Measure of Human Capital: Evidence from LinkedIn Profiles”: LinkedIn skill sets are meaningfully related to education, work experience, and labor-market outcomes, supporting a deliberate, role-relevant skills strategy.
